# 财签小助手 - NFC 求签游戏

一个现代简约风格的财运求签小游戏，支持碰一碰 NFC 快速打开。

## 功能特点

- 🎨 现代简约设计风格
- 🔮 摇签筒动画（4.5秒）
- 💰 12支财运签文，涵盖各种运势
- 📱 移动端优化，体验流畅
- 🏷️ 支持NFC标签写入

## 文件说明

- `index.html` - 网页主文件
- `style.css` - 样式文件
- `script.js` - 游戏逻辑

## 使用方法

### 本地测试
直接用浏览器打开 `index.html` 即可运行。

### 部署到网上（GitHub Pages）

1. 注册 GitHub 账号：https://github.com
2. 创建新仓库，命名为 `yourname.github.io`（把 yourname 换成你的用户名）
3. 将三个文件上传到仓库
4. 等待1-2分钟，访问 `https://yourname.github.io` 即可

### 写入 NFC 标签

1. 下载 NFC Tools App（手机应用商店搜索）
2. 打开 App → 写入 → URI
3. 粘贴你的网页链接（如 `https://yourname.github.io`）
4. 将 NFC 贴纸贴近手机，完成写入

## 签文列表

包含12支财运签文：
- 大吉、上上签、上签等不同级别
- 涵盖财富运势、投资建议、理财提示
- 每次随机抽取，体验新鲜

## 技术栈

- 纯 HTML + CSS + JavaScript
- 无需任何框架
- 响应式设计，适配所有设备
